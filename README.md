## Resume Rx

**A streamlined platform for building professional, ATS-friendly resumes.**

Resume Rx is a web-based application designed to simplify the resume-creation process. It focuses on clean typography, structured layouts, and real-time editing to help users generate high-quality professional documents efficiently.

---

### Key Features

* **Real-time Preview:** View changes instantly as you input your professional history.
* **Modular Design:** Easily add, remove, or reorder sections such as Experience, Education, and Projects.
* **ATS Optimization:** Layouts are structured to be easily parsed by Applicant Tracking Systems.
* **Modern UI:** A clean, distraction-free interface for data entry.
* **PDF Export:** High-quality PDF generation for immediate job applications.

---

### Tech Stack

* **Framework:** React / Next.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **PDF Generation:** React-PDF or equivalent libraries

---

### Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**
```bash
git clone https://github.com/Gxgxn/resume-rx.git
cd resume-rx

```


2. **Install dependencies**
```bash
npm install

```


3. **Run the development server**
```bash
npm run dev

```


4. **Access the application**
Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) in your browser to view the app.

---

### Project Structure

* `/components` - Reusable UI elements (inputs, buttons, cards).
* `/templates` - Different resume layout designs.
* `/store` - State logic for handling user data.
* `/styles` - Global CSS and Tailwind configurations.
